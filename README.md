# redux-action-types README

redux action types 自动化工具

## Features
在编写redux action types 时，自动生成值,在js或ts文件中使用`// enableAutoFixActionTypes`来启用功能
```js
// enableAutoFixActionTypes

const AUTH_LOGIN = "";
//   |
//   ∨
const AUTH_LOGIN = "auth_login";
```
